Required Software
1. Google Chrome
2. Python 2 or 3

Instructions
1. (Windows) Append the path to the Python directory with python.exe to the PATH variable in Environment Variables.
2. (Windows) Click simple-http-server-windows.bat.
3. (Linux) At this directory, type ./simple-http-server-linux.sh into the Terminal.
4. Type localhost:8000/<path to this folder> on Chrome.
5. Comment and/or uncomment labelled lines in script.js to customise the Progress Bar.
